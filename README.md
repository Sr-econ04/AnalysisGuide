# 分析手法ガイド

**URL：https://sr-econ04.github.io/AnalysisGuide/index.html**

## このツールについて

計量経済学をまだ学び始めたばかりの学部生が、卒業研究で「どの分析手法を使えばいいかわからない」「結果の数字の意味がわからない」という場面で参照することを想定して作成しました。

**現在β版です。** 内容に誤りや不足がある可能性があります。気づいた点はIssueやPull Requestでお知らせいただけると助かります。

## できること

- 質問に答えると、データの特性に合った分析手法を提案する
- 各手法の結果の読み方・確認すべきこと・使ってはいけない場合を確認できる
- 手法名から直接検索して詳細を調べられる

## できないこと

- 研究テーマの設定・データ収集・論文の書き方はカバーしていません
- 分析の妥当性や解釈の判断は、指導教員への確認が必要です
- このツールはあくまで補助であり、手法を選んだ理由を自分の言葉で説明できることが重要です

## 収録手法（25手法）

| カテゴリ | 手法 |
|---|---|
| 連続値 | OLS線形回帰・対数線形回帰・分位点回帰・トービット回帰・ヘックマン選択モデル |
| カウント | ポアソン回帰・負の二項回帰・ZIP・ZINB・ゼロ切断ポアソン回帰 |
| 2値・カテゴリ | ロジット・プロビット・順序ロジット・多項ロジット・混合ロジット |
| 生存分析 | コックス比例ハザードモデル |
| パネルデータ | 固定効果モデル・変量効果モデル・ハウスマン検定 |
| 因果推論 | DID・操作変数法・RDD・PSM・IVプロビット・合成コントロール法 |
